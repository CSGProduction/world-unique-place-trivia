# 🌍 Teka-Teki Dunia: 100 Level

![Version](https://img.shields.io/badge/version-1.0.0-green.svg)
![Tech](https://img.shields.io/badge/tech-HTML--CSS--JS-orange.svg)

**Teka-Teki Dunia** adalah sebuah game edukasi berbasis web yang menantang wawasan geografi kamu! Jelajahi 100 tempat ikonik, candi bersejarah, hingga gedung pencakar langit yang tersebar di benua **Asia**, **Eropa**, dan **Timur Tengah**.

## 🚀 Fitur Utama
- **100 Level Menantang**: Dari Borobudur hingga Menara Eiffel.
- **Sistem Kategori**: Level dibagi berdasarkan wilayah (Asia, Eropa, Timur Tengah).
- **Sistem Hint (Petunjuk)**: Terjebak di satu level? Gunakan hint untuk memunculkan huruf jawaban.
- **Skor Dinamis**: Dapatkan +20 poin untuk setiap jawaban benar.
- **Responsive Design**: Tampilan yang nyaman dimainkan di HP maupun Laptop.

## 🛠️ Teknologi yang Digunakan
Proyek ini dibangun murni menggunakan **Vanilla Web Stack**:
- **HTML5**: Untuk struktur utama game.
- **CSS3**: Untuk desain antarmuka (UI) bertema gelap (*dark-mode*) yang modern.
- **JavaScript (ES6)**: Untuk logika permainan, sistem skor, dan manajemen level.

## 🎮 Cara Bermain
1. Buka file `index.html` di browser pilihan kamu.
2. Perhatikan petunjuk teks dan nama negara yang muncul.
3. Ketik jawaban pada kolom teks yang tersedia.
4. Klik tombol "Jawab!" atau tekan Enter.
5. Gunakan tombol **💡 Hint** jika kesulitan (akan memotong 5 poin skor kamu).

## 📂 Cara Instalasi (Lokal)
1. Download atau Clone repository ini.
2. Pastikan file HTML, CSS, dan JS berada dalam satu folder.
3. Klik kanan pada `index.html` dan pilih **Open with Browser**.
